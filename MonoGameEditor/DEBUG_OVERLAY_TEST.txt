# Debug: Teste Manual do Overlay

Execute este código no Immediate Window do Visual Studio para testar o overlay manualmente:

```csharp
var mainWindow = System.Windows.Application.Current.MainWindow as MonoGameEditor.MainWindow;
mainWindow?.ShowLoadingOverlay("TESTE MANUAL");
```

Espere 5 segundos e depois execute:

```csharp
var mainWindow = System.Windows.Application.Current.MainWindow as MonoGameEditor.MainWindow;
mainWindow?.HideLoadingOverlay();
```

Se isso funcionar, o overlay está OK mas não está sendo chamado corretamente.
Se não funcionar, há problema no XAML ou na estrutura.
